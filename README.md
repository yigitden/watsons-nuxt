# Watsons LabTurkiye
Demo: https://watsons-lab.netlify.app/

 

![Web - Mobile](previews/labturkiye.gif)

## Technologies Used

- NuxtJS 
- Tailwind
- Nuxt Agile
- Axios

## Installation

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
 
```
